Pourquoi un site ?

Après la création du BomberMan, j'ai voulu mettre en place une plateforme qui me permettrait de mettre à disposition mes projets. L'idée du site web est venue très naturellement.

Les technos

Le site est intégralement développé en HTML, CSS, JS, PHP et SQL pour la version actuelle. De même que pour BomberMan j'ai développé ce site en autodidacte je n'ai pas eu l'occasion de le refaire depuis.
