Bomber Man

INFO

Projet réalisé avant mon entré à l'IUT

année de réalisation: 2019

Ce projet s’intitule BomberMan , il reprend le nom du célèbre jeu dont il est inspiré. Créer de manière totalement autodidacte à l’aide de tutoriels trouvé sur internet, ce projet m’a permis d’appréhender la programmation de façon amusante et motivante. Réaliser durant mon temps libre j’ai mis 8 mois pour lui donner vie.

Les technos

Le projet est développé à l’aide d’Unity 3D et en langage C#. Unity est un moteur de jeu gratuit pour les petits studios et les développeurs indépendants, de plus, il offre un store d'asset très complet d'où le choix de l'utiliser pour ce projet. Pour ce qui est du langage, C# étant le langage utilisé dans Unity, son utilisation était évidente.

Lors de ce projet Visual Studio était le seul IDE que je connaissais et les tutoriels que je regardais le recommandait.

Ressenti / Feedback

Ce projet a une valeur très importante pour moi, il a confirmé ma nouvelle orientation scolaire et professionnelle. Il me permet aussi d'avoir une rétrospective sur le chemin parcouru depuis que j'ai commencé la programmation.

Règle du jeu

Bomber Man est un jeu plutôt simple, l'objectif est de tuer les joueurs des autres équipes. Pour y parvenir, le joueur possède un stock de bombe qu'il doit utiliser pour créer un passage à travers la carte pour atteindre les autres joueurs. Pour gagner il suffit d'être le dernier joueur en vie.

Element de la carte

La carte a été réalisé par mes soins. Les blocs de brique sont ceux dont la destruction est possible pour créer son chemin quant aux blocs de bedrock ils sont indestructibles.

Les briques et la debrock sont des textures reprises du célèbre jeu Minecraft.
