Planning BTP

INFO

Projet réalisé en 2ème année de BUT.
Retrouvez le projet sur mon GitHub: Planning BTP

année de réalisation: 2023

Planning BTP est une application web destiner à une entreprise de BTP voulant gérer ses chantiers, l'utilisation de ses engins et le matériel utiliser sur chacun d'eux. De plus l'application permet d'enregistrer les frais liés aux chantiers.

Réalisé dans le cadre des projets tutoré, proposé par la formation du BUT. Le projet a été réalisé en groupe de 3 personnes dont moi. Une des professeurs du département informatique madame Amandine Berger, m'a proposé de travailler sur son projet.

Les technos

Ce projet est développé en HTML, PHP, CSS et SQL. Lors du commencement de ce projet, les cours de développement PHP venaient de m'être dispensé. De plus le seul langage disponible pour l'utilisation de base de données était le SQL, d'où le choix de mon groupe de travailler avec ces langages.

Consignes du projet

L'objectif de ce projet était d'avoir une application web, pour être consulter sur tout types de support. Les points clés du projet était les suivants

  • Pouvoir gérer les chantiers
  • Pouvoir assigné du matériel, du personnel, des véhicules à différents chantiers
  • Permettre d'enregistrer les heures des employés
  • Permettre d'enregistrer les dépenses lié à un chantier
